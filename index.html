HTML Cheat Sheet

Introduction

This page covers the key elements of HTML and HTML5 using pure markup. Each section lists core tags with short explanations.

Document Structure

Note: Basic layout of an HTML document.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Title</title>
  </head>
  <body>
    <!-- content -->
  </body>
</html>

Metadata

Note: Elements in the <head>.

  • <meta charset="utf-8"> – character encoding.
  • <meta name="viewport" content="width=device-width,initial-scale=1">
  • <title> – document title.
  • <link> – external resource link.
  • <style> – internal CSS (omitted here).
  • <base> – base URL for relative links.

Text Basics

Note: Headings and paragraphs.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>Paragraph text.</p>

Inline Text Semantics

Note: Mark up words or phrases.

  • <em> emphasis
  • <strong> strong importance
  • <small> small print
  • <mark> highlight
  • <code> inline code
  • <abbr title="abbreviation">abbr</abbr>
  • <time datetime="2024-01-01"> date or time

Grouping Content

Note: Block-level elements.

  • <div> generic container.
  • <p> paragraph.
  • <hr> thematic break.
  • <blockquote> quoted section.
  • <pre> preformatted text.

Lists

Note: Ordered, unordered, and description lists.

<ul>
  <li>Item</li>
</ul>
<ol>
  <li>First</li>
</ol>
<dl>
  <dt>Term</dt>
  <dd>Definition</dd>
</dl>

Links

Note: Hyperlinks and anchors.

<a href="https://example.com" target="_blank">Example</a>
<a id="top"></a> <a href="#top">Internal link</a>

Images

Note: Embedding images.

<img src="path/to/image.jpg" alt="description" width="200" height="100">

Audio and Video

Note: Media elements.

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
</audio>
<video controls width="320" height="240">
  <source src="movie.mp4" type="video/mp4">
</video>

Tables

Note: Displaying tabular data.

<table>
  <caption>Data</caption>
  <thead><tr><th>Header</th></tr></thead>
  <tbody><tr><td>Cell</td></tr></tbody>
</table>

Forms

Note: Collecting user input.

<form action="/submit" method="post">
  <label for="name">Name</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Send">
</form>
  • Common controls: <input>, <textarea>, <select>, <button>

Semantic Elements

Note: Meaningful page structure.

  • <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • <figure> with <figcaption>

Embedded Content

Note: External resources.

<iframe src="https://example.com" title="Example"></iframe>
<embed src="file.swf">
<object data="file.pdf" type="application/pdf"></object>

Scripting

Note: Adding JavaScript. Example only—no scripts run here.

<script>
  console.log('Hello');
</script>

Global Attributes

Note: Attributes usable on most elements.

  • id, class, style (inline CSS), title
  • hidden, tabindex, accesskey, draggable
  • data-* custom data attributes

Accessibility

Note: ARIA and best practices.

  • Use semantic elements for structure.
  • alt text on images.
  • ARIA roles: role="navigation", etc.
  • Label form controls with <label>.

Best Practices

Note: General tips.

  • Keep markup clean and well-indented.
  • Use semantic elements for clarity.
  • Include lang attribute on <html>.
  • Avoid deprecated tags like <font>.
